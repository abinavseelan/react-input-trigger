## Usage

### Getting Started

- Import the component from the package.

```js
import InputTrigger from 'react-input-trigger';
```

- Wrap your existing `<textarea />` or `<input />` element with `<InputTrigger />`

```jsx
<InputTrigger>
  <textarea />
</InputTrigger>
```
